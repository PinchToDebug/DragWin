## ✨ 기능

- 마우스 오른쪽 버튼으로 창 이동
- 측면 및 모서리에서 창 크기 조정
- 스크롤 시작 창 뒤에 있는 창 스크롤 _(중간 버튼 누른 상태로 스크롤)_
- 상단 모서리에서 가상 데스크톱 간 스크롤
- FancyZones 자동 호출 옵션
- 다중 디스플레이 지원
- 스크롤 휠 드래그로 가상 데스크톱 전환
- Win + 스크롤로 가상 데스크톱 전환
- Alt 키 누른 상태에서 창 투명도 변경 옵션
- 다국어 지원
- 전체 화면 애플리케이션에 영향 없음
- 마우스 오른쪽 클릭에 영향 없음
- 드래그 제외 옵션
